NodeJs 简介

Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine.

Node.js®是基于Chrome V8 JavaScript引擎构建的JavaScript运行时。

NodeJs 简介

Node.js 是能够在服务器端运行JavaScript 的开放源代码、跨平台 JavaScript 运行环境。Node.js 由 Node.js Foundation持有和维护，亦为Linux 基金会的项目
